# W1M1 - mtcars 데이터 분석 및 시각화

## 학습 목표
- mtcars 데이터셋을 분석하고 그래프를 그리는 방법을 배운다.
- Jupyter notebook의 기본 사용법을 배운다.
- pandas의 DataFrame 기본 사용법을 배운다.
- matplotlib로 그래프를 그리는 방법을 배운다.

## 사전지식
데이터셋의 features:

| 컬럼 | 설명 |
| --- | --- |
| mpg | Miles/(US) gallon |
| cyl | Number of cylinders |
| disp | Displacement (cu.in.) |
| hp | Gross horsepower |
| drat | Rear axle ratio |
| wt | Weight (lb/1000) |
| qsec | 1/4 mile time |
| vs | V/S |
| am | Transmission (0 = automatic, 1 = manual) |
| gear | Number of forward gears |
| carb | Number of carburetors |

데이터 파일: [mtcars.csv](mtcars.csv)

## 기능 요구사항
- Jupyter notebook을 만든다.
- 다음 DataFrame methods로 데이터셋 결과를 출력한다: `head`, `tail`, `shape`, `columns`, `info`, `dtypes`, `describe`
  - `shape`로 row count 출력
  - 이름이 없는 컬럼을 찾아 적합한 이름으로 변경
- `gear`, `transmission` features의 값 종류 개수를 출력한다.
- gear와 transmission의 조합별 차량 개수를 알아낸다.
  - `pandas.crosstab`을 사용하는 방법과 사용하지 않는 방법 모두로 결과를 출력한다.
  - 결과를 화면에 출력한 뒤, 바 그래프로 그린다.
    - 타이틀: `# of Cars by Gear x Transmission`
    - x축 레이블: `(Gear, Auto Transmission)`
    - y축 레이블: `# of Cars`
- 모든 features의 히스토그램을 그린다. (타이틀: `Histograms of Variables`, 보기 좋게 배치)
- Cylinder vs HorsePower의 상관 관계를 scatter 차트로 그린다. (타이틀: `Cylinder vs Horse Power`)
- MPG vs HorsePower의 상관 관계를 scatter 차트로 그린다. (타이틀: `MPG vs Horse Power`)
- 각 변수 간 상관계수 표를 출력한다.

## 프로그래밍 요구사항
- 데이터는 pandas DataFrame으로 처리한다.
- 가급적 numpy는 사용하지 않는다.
- 그래프는 matplotlib를 사용한다. (필요 시 다른 라이브러리 허용)

## 팀 활동 요구사항
- 이 데이터셋을 분석해서 얻을 수 있는 경제적 가치는 무엇인지, 어떤 비즈니스 상황에서 어떤 가치를 얻을 수 있는지 토의한다.
- 상관 관계가 높은 변수 조합 2개를 임의로 선택해 그래프로 그리고 결론을 토의한다.
