# W1M2 - SQL 기초

## 학습 목표
- SQL의 기초에 대해서 공부한다.
- Tutorial의 Home에서부터 Operators까지 읽고 Example을 실행한다.
- Jupyter Notebook에서 SQL을 사용해서 데이터를 처리하는 것을 공부한다.

## 기능 요구사항
- Jupyter Notebook을 만든다.
- SQL Tutorials에서 Example을 하나씩 수행하고 그 결과를 노트북에 출력한다.
- 원하는 출력결과를 얻기 위해 필요하다면 적합한 Demo Table을 만든다.

## 프로그래밍 요구사항
- sqlite3 라이브러리를 사용한다.
- pandas와 같은 라이브러리는 사용해서는 안 된다. SQL 명령어로만 원하는 결과를 얻어야 한다.

## 예상결과 및 동작예시
- Query의 결과가 출력된 Jupyter notebook을 제출한다.

## 팀 활동 요구사항
- 각자가 이해하기 어려웠던, 또는 이해하지 못한 keyword에 대해서 함께 토의한다.
- 해당 내용을 팀위키에 기록하고 제출한다.
