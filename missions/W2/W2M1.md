# W2M1 - multiprocessing.Pool

## 학습 목표
- 파이썬의 `multiprocessing.Pool`에 대한 이해도를 보여주기 위해, 워커 풀을 활용하여 작업을 동시에 실행하는 스크립트를 작성한다.

## 기능 요구사항
- 작업 정의: 각 작업에 이름(예: 'A', 'B' 등)과 소요 시간(초 단위의 정수)을 부여하여 작업 목록을 정의한다.
- 워커 풀 설정: 지정된 수(이 경우 2)의 작업자로 풀을 초기화한다.
- 작업 실행: 작업 목록의 각 작업에 대해
  - 작업 이름과 소요 시간을 나타내는 대기 메시지를 출력한다.
  - `time.sleep`을 사용하여 지정된 시간 동안 잠을 자면서 작업을 시뮬레이션한다.
  - 작업을 완료한 후, 작업이 완료되었음을 알리는 메시지를 출력한다.
- 동시성: 워커 풀(Pool)을 사용하여 작업이 동시에 실행되도록 한다.

## 프로그래밍 요구사항
- 파이썬의 `multiprocessing` 모듈을 사용한다.
- 각 작업에 이름과 소요 기간이 포함된 작업 목록을 정의한다.
- 멀티프로세싱 풀을 사용하여 작업을 동시에 실행한다.
- 각 작업을 시뮬레이션하기 위해 대기 메시지를 출력한 후 지정된 시간 동안 대기하는 함수(`work_log`)를 정의한다.
- `Pool`의 `map` 함수를 사용하여 `work_log` 함수를 작업 목록에 매핑한다.

## 예상결과 및 동작예시
```
프로세스 A 대기 시간 5초,
프로세스 B 대기 시간 2초,
프로세스 B 완료.
프로세스 C 대기 시간 1초,
프로세스 C 완료.
프로세스 D 대기 시간 3초,
프로세스 A 완료.
프로세스 D 완료.
```
