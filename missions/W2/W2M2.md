# W2M2 - multiprocessing.Process

## 학습 목표
- 대륙의 이름을 출력하는 여러 개의 프로세스를 생성하는 스크립트를 작성하여, 파이썬의 `multiprocessing.Process`에 대한 이해도를 보여준다.

## 기능 요구사항
- 대륙의 이름을 포함한 문장을 출력하는 함수를 만든다.
- 다른 이름이 주어지지 않으면 함수는 기본 대륙으로 "Asia"를 사용해야 한다.
- 스크립트는 기본 대륙으로 함수를 한 번 실행하고, America, Europe, Africa라는 서로 다른 대륙 이름으로 세 번 더 실행해야 한다.
- 스크립트는 종료되기 전에 함수의 모든 인스턴스가 실행을 완료하도록 보장해야 한다.

## 프로그래밍 요구사항
- 파이썬의 `multiprocessing` 모듈을 사용한다.
- 주어진 대륙 이름을 출력하는 함수를 정의한다.
- `Process` 클래스를 사용하여 이 함수를 호출하는 여러 프로세스를 생성한다.
- 모든 프로세스가 제대로 시작되고 완료되도록 보장한다.

## 예상결과 및 동작예시
```
The name of continent is : America
The name of continent is : Europe
The name of continent is : Asia
The name of continent is : Africa
```
