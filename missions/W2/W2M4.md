# W2M4 - multiprocessing (Queue + Process)

## 학습 목표
- 큐(Queue)와 프로세스(Process)를 사용하여 동시에 여러 작업을 처리하는 파이썬 `multiprocessing` 모듈에 대한 이해도를 보여준다.

## 기능 요구사항
- 작업 분배:
  - 10개의 작업(Task no 0 ~ Task no 9)을 큐(`tasks_to_accomplish`)를 이용해 여러 프로세스가 실행하도록 분배한다.
- 프로세스 실행:
  - 4개의 프로세스를 생성하여 `tasks_to_accomplish`에서 동시에 작업을 가져와 실행하고, 결과를 다른 큐(`tasks_that_are_done`)에 저장한다.
  - 각 프로세스는 현재 실행 중인 작업을 출력하고, 완료 메시지를 `tasks_that_are_done`에 추가해야 한다.
  - `time.sleep(0.5)`를 사용하여 작업 실행 시간을 시뮬레이션한다.
- 작업 완료:
  - `Process.join()`을 사용하여 모든 프로세스가 작업을 완료하고 동기화되도록 보장한다.
  - 모든 프로세스가 종료된 후 `tasks_that_are_done`에 저장된 완료 메시지를 출력한다.

## 프로그래밍 요구사항
- 모든 작업을 실행하는 프로그램을 작성하고 'multiprocessing_all_in_one.py'로 이름 붙인다.
- 파이썬의 `multiprocessing` 모듈을 사용한다.
- 작업과 결과를 위해 `Queue`를 사용한 작업 분배 시스템을 구현한다.
- 여러 프로세스를 생성하여 작업을 동시에 실행한다.
- `Queue.get_nowait()`와 예외 처리를 사용하여 프로세스 간 작업을 동적으로 관리한다.
