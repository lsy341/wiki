# W2M5 - 워드클라우드

## 학습 목표
- 레이블링된 텍스트 데이터를 시각화하는 방법을 배운다.
- 작은 데이터셋을 직접 만들어서 전체 과정을 체험해 본다.

## 사전지식
- 데이터셋은 160만 개의 트윗으로 구성되어 있으며, 어떤 컬럼에도 결측값이 없다.
- 각 트윗은 감성 타겟(0: 부정, 2: 중립, 4: 긍정), ID, 타임스탬프, flag(query 또는 'NO_QUERY'), 사용자명, 텍스트를 가진다.
- 감성 타겟은 균형을 이루고 있으며, 긍정과 부정 레이블의 개수가 동일하다.

## 기능요구사항
- Jupyter notebook에 프로세스 과정을 출력하고 결과 그래프도 출력한다.
- 긍정 감성, 부정 감성에 각각 해당하는 단어들을 모아서 워드클라우드를 화면에 표시한다.
- 워드클라우드를 만들 때 사용할 단어들은 샘플링을 통해서 선택한다.
- 하나의 워드클라우드에는 최대 200개의 단어만 표시한다.

## 프로그래밍 요구사항
- pandas 라이브러리를 사용한다.
- matplotlib 라이브러리를 사용한다.
- wordcloud 라이브러리를 사용한다.
- 2개의 워드클라우드는 하나의 plot에 그린다.

## 팀 활동 요구사항
- 제공된 데이터셋과 유사한 데이터셋을 이용해서 비즈니스 가치를 만드는 프로젝트를 진행해야 한다.
- 먼저 해당 데이터셋을 통해서 어떤 비즈니스 가치를 만들 수 있을지에 대해 토론한다. (먼저 여러 개의 아이디어를 만들고 그중에 하나를 고르는 게 효율적이다.)
- 데이터셋은 웹 스크레이핑을 통해 직접 만든 다음, 워드클라우드를 만들어 본다.
- 데이터셋을 만들 때 어떤 작업들이 추가적으로 필요할지 생각해본다.
- 프로토타이핑에는 최소 1,000개 이상의 데이터를 사용한다.
- 프로토타입을 만든 다음, 만들기 전에 생각했던 비즈니스 가치를 만들 수 있는지에 대해서 다시 토론한다.
- 팀 위키에 프로젝트 과정 전체를 정리해서 제출한다.

## 추가학습거리
- Predicting Election Results from Twitter Using Machine Learning Algorithms
- 아래의 Summary를 읽고 소감을 본인의 위키에 기술한다.

> Yadala, Sucharitha & Vijayalata, Y. & Prasad, V. (2020). Predicting Election Results from Twitter Using Machine Learning Algorithms. Recent Advances in Computer Science and Communications. 13. 10.2174/2666255813999200729164142.

**서론**
오늘날 소셜 미디어 네트워크는 개인 간에 정보를 공유하는 데 중요한 역할을 하고 있다. 여기에는 현재 실세계에서 일어나고 있는 뉴스와 이벤트에 관한 정보도 포함된다. 선거 결과를 예측하는 것은 소셜 미디어를 통해 흥미로운 연구 주제로 떠오르고 있다. 이 논문에서는 마이크로블로그에서 서브 이벤트 탐지와 감성 분석을 결합하여, 소셜 미디어 사용자들이 드러내는 정치적 성향을 분석하고 시각화함으로써 선거 결과를 예측하는 전략을 제안한다.

**방법론**
이 접근법은 마이크로블로그에서 감성 데이터를 발굴하고 분석하여 후보자의 인기도를 예측한다. 일반적으로 많은 기관과 언론사들은 사전 여론조사와 전문가 의견을 바탕으로 선거 결과를 예측하지만, 본 모델에서는 트위터 데이터를 수집하고 후보자에 대한 트윗의 감성을 분석하여 선거 결과를 예측하는 데 사용한다.

**결과**
1, 2, 3위 정당이 2019년 AP 주의회 선거에서 얻은 의석 수는 각 정당의 PSS를 식(2), (3), (4)를 이용해 산출하여 결정했다. 표 2에서 실제 선거 결과와 모델의 예측 결과를 비교했으며, 그 결과가 실제 결과와 매우 근접했다. 감성 극성 분류에는 SVM과 15-fold 교차 검증을 사용했고, 학습 데이터셋 기준 94.2%의 정밀도를 얻었다. 학습 데이터셋은 총 7,500개의 트윗(긍정 3,750개, 부정 3,750개)으로 구성된다.

**결론**
본 연구 결과는 제안한 모델이 기존 베이스라인 대비 94.2%의 정확도로 선거 결과를 정밀하게 예측할 수 있음을 보여준다. 실험 결과는 실제 선거 결과와 매우 근접했으며, 출구조사나 결과 승인을 위해 여러 여론조사 기관이 사용하는 전통적인 방식과 비교했을 때, 소셜 미디어 데이터가 더 나은 정확도로 예측할 수 있음을 보여주었다.

**논의**
향후에는 트위터가 정치 활동 수단으로서 인기를 얻고 있고, 정치인과 개인들이 정치적 소통과 정보 교류를 위해 마이크로블로그로 눈을 돌리고 있는 다른 지역과 국가로 이 연구를 확장하고자 한다. 또한 일반 선거와 정치인뿐 아니라, 국가 기관을 포함한 다양한 분야로도 이 연구를 확장하고자 한다.
