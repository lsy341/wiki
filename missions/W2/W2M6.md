# W2M6 - 데이터 프로덕트 배포

## 학습 목표
- 당신의 첫 데이터 프로덕트(프로토타입)를 배포한다.
- 로컬에서 docker container image를 만들어서 AWS EC2에 배포해 본다.
- 자신이 만든 데이터 프로덕트를 다른 사람이 사용할 수 있도록 배포한다.

## 기능요구사항
- AWS EC2에 배포된 후에 서버 주소(public DNS name)로 들어가면 JupyterLab 인터페이스가 실행되어야 한다.
- 해당 화면에서 W1에서 만들었던 Jupyter notebook을 선택해서 실행할 수 있어야 한다.

## 프로그래밍 요구사항
- AWS 계정 만들기
  - gmail에서 새로운 이메일을 생성한다. (추천: `aws.dano@gmail.com`처럼 `aws.`을 prefix로 사용하는 계정을 만들어서 사용하면 편하다.)
  - 새로 생성한 이메일로 aws 계정을 생성한다. 새로운 계정에서 free-tier EC2 인스턴스를 12개월 동안 무료로 사용할 수 있다.
- AWS free-tier EC2 생성하기
  - free-tier EC2 인스턴스를 생성한다.
  - 반드시 User-data를 사용해서 Docker를 설치하고 구성한다. (코드 제출할 때 반드시 `user-data.sh`를 포함한다.)
- Docker Image 생성하기
  - Local에서 Docker Desktop을 이용해서 AWS EC2에 배포할 docker container image를 생성한다.
  - 어떤 OS를 선택해야 할까?
  - 어떤 소프트웨어를 설치해야 할까?
  - 어떤 파일을 담아야 할까?
- Docker Image 배포하기
  - AWS ECR(Elastic Container Registry)에 Docker container image를 push한다.
  - EC2에서 container image를 pull한다.
